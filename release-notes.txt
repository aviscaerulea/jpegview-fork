## 主な変更点

- HEIC 内部デコード対応（libheif）、デコード失敗時は WIC フォールバック
- AVX2 命令セット有効化（x64）
- パフォーマンス最適化
  - スレッド数上限を 16→64 に拡大（ProcessingThreadPool）
  - AVIF デコーダのスレッド数を INI 設定値に変更
  - mimalloc メモリアロケータを統合（マルチスレッド環境でのメモリ割り当て最適化）
  - AlphaBlendBackground 整数演算化
- 物理コア数検出を Windows API に変更（AMD Ryzen 対応）
- About ダイアログ改善（表示領域拡大、README リンク対応、fork 版 URL 追加）
- INI デフォルト値変更（IniEditor→system、NavigateWithMouseWheel→true）
- サードパーティライブラリを最新安定版に更新（セキュリティ修正含む）
