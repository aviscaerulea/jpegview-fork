# v1.3.46.0-20260215.1

## 概要

VC++ 再頒布パッケージ未インストール環境での互換性を改善し、リポジトリ名称を JPEGView-nt に変更したリリース。

## 新機能

- **MSVC ランタイム DLL の同梱**
  - プリビルト DLL（PDFium 等）が必要とする MSVC ランタイム DLL（msvcp140.dll、vcruntime140.dll、vcruntime140_1.dll）を配布パッケージに同梱
  - VC++ 再頒布パッケージが未インストールの環境でも JPEG XL、HEIF、AVIF、RAW、PDF 形式を正常に読み込めるように改善

## 改善

- **リポジトリ・アプリ名称変更**
  - JPEGView-fork から JPEGView-nt に名称変更
  - タイトルバーに "JPEGView-nt" を表示
  - About ダイアログの URL を更新
- **リリース自動化スクリプトの追加**
  - release.ps1 による一貫したリリースフローの自動化
  - Taskfile.yml にビルド・リリースタスクを追加
- **INI 設定のデフォルト値変更**
  - `IniEditor` のデフォルト値を `system` に変更（Windows 既定のテキストエディタで開く）
  - `NavigateWithMouseWheel` のデフォルト値を `true` に変更（マウスホイールでの画像切り替えを有効化）

## 修正

- zip 入れ子問題の修正（配布 zip 内に不要な入れ子ディレクトリが作成される問題を解消）
- README の更新（リポジトリ名称変更に伴う修正）
- .gitignore にビルド成果物を追加
- HEIFWrapper.cpp のフォーマット修正（ファイル末尾改行）
